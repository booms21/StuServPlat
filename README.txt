主题：学生综合服务平台Demo

我们是以广大学生的角度出发制作的简单、创意、美观的综合多元化服务的校园便捷平台，通过该平台能对校园一卡通的充值、学生成绩的查询、校园的动态、查看学生们精彩的瞬间等功能。


运行条件：

电脑分辨率：1366 ×768 及以上（注：建议使用正常的pc屏幕分辨率）

兼容浏览器：谷歌浏览器51.0（推荐）、火狐47.0、IE9以上、360浏览器8.1、QQ浏览器9.4.1


使用方法：

运行网站根目录下的index.html文件（建议使用推荐浏览器运行）

进入网站后4个彩色大按钮上方是网站头部和实事焦点图，点击焦点图左右两边的箭头或右下的白点对焦点图的切换，而下中间部分方是主要内容的可变显示区，首页的中间部分是显示校园的最近的几条动态新闻，用户可点击蓝色的显示全部进入查看详细页面 ，中间部分的右边有快捷通道，如在您浏览其他内容的时候可以随时对一卡通等进行充值，校园一卡通充值需先选择所在的学校

树洞功能为平台用户的留言功能，留言功能在交流空间 - 树洞

交流空间-学生风采功能 ，用户可点击图片进行大图观看。
